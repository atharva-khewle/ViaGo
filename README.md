## 🤘 What even is this?

ViaGo is the **travel booking megathread** you wish you already had.  
We handpicked and organised every site that:

- **Books** your flight, bus, train, hotel, cruise, car, insurance, eSIM, airport service…
- **Compares** prices across all the bookers so you don’t get ripped off.

No reviews. No blog spam. Just **clean links that get you to the booking faster**.

Built by travellers, for travellers. Completely free.  
Go ahead — **ViaGo**.

## 🔥 Quick Start (Run it locally)

### 1. Grab the magic
```bash
git clone https://github.com/atharva-khewle/ViaGo.git
cd ViaGo
```

### 2. Install everything
```bash
npm install
```

### 3. Fire it up
```bash
npm run dev
```

Open `http://localhost:3000` and boom — all the travel links you’ll ever need, right on your screen.

*Note: Firebase is wired but sleeping. The site runs on static data right now, ready to switch to Firestore with zero refactoring. You’ll find a neat firebase.js placeholder and a drop-in query function in the category page.*

## 🛠️ Built With
- **Next.js 14+ (App Router)** – fast, modern, SEO-friendly
